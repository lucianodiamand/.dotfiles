* dotfiles
My personal dot files.

* Dependencies
** Nerd fonts
Download the fonts:
#+begin_src sh
wget -c https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/Hack.zip -P /tmp
#+end_src

Uncompress into directory: ~/.local/share/fonts

Run:
#+begin_src sh
fc-cache -fv
#+end_src

** typescript-language-server
To install:
#+begin_src sh
sudo npm i -g typescript-language-server
#+end_src

** Packer
To install packer:
#+begin_src sh
git clone --depth 1 https://github.com/wbthomason/packer.nvim ~/.local/share/nvim/site/pack/packer/start/packer.nvim
#+end_src

* Platforms
** Debian
Install common dependencies:
#+begin_src sh
sudo apt install stow git curl unzip fontconfig
#+end_src

Note: lua-language-server needs to be compiled from source on Debian (TODO).

** NixOS
Use the Home Manager flake in `nixos/`:
#+begin_src sh
home-manager switch --flake ./nixos
#+end_src

** FreeBSD
Install dependencies with your package manager (pkg), then run stow as below.
